## Getting Started  
To get a local copy up and running follow these simple steps.  

### Prerequisites  
- Node.js  
- npm  

### Installation  
1. Clone the repo  
   ```sh  
   git clone https://github.com/Zyrova/frontend.git  
   ```  
2. Install NPM packages  
   ```sh  
   npm install  
   ```  
3. Run the application  
   ```sh  
   npm start  
   ```
